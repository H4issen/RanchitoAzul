# Ranchito Azul — Sistema de Gestión para Restaurante

¿Tienes un negocio gastronómico y aún manejas pedidos en papel, precios en pizarra y clientes en una libreta? Este sistema web unifica todo en un solo lugar: tu carta digital, tus pedidos, tus clientes y el control de tu negocio, accesible desde cualquier dispositivo.

## ¿Qué incluye?

### 🧑‍🍳 Para tus clientes
Una página web profesional donde ven la carta completa con precios, fotos y descripciones. Pueden armar su pedido desde el celular mientras están en la mesa o desde su casa. Sin apps, sin vueltas.

### 📊 Para ti (el dueño o administrador)
Un panel privado con todo lo que necesitas para operar el negocio:

- **Menú digital** — Agrega, edita o elimina productos al instante. Cambia precios, ofertas y categorías sin esperar a un programador.
- **Banners y promociones** — Administra las imágenes del slider principal para destacar combos y lanzamientos.
- **Control de usuarios** — Crea cuentas para tu personal (administradores) y lleva registro de tus clientes. Puedes activar o desactivar accesos cuando quieras.
- **Pedidos** — Visualiza y gestiona las órdenes entrantes.
- **Configuración del negocio** — Cambia el horario, teléfono, dirección, enlaces a redes sociales y mapa sin tocar una línea de código. También activa un modo "mantenimiento" para cuando estés actualizando.
- **Tema personalizable** — Elige entre modo claro u oscuro y color base (azul, verde, morado, naranja, rojo, teal).

### 🔐 Seguridad desde el inicio
- Acceso por roles (administrador / cliente)
- Contraseñas protegidas con encriptación
- Cada administrador se puede desactivar individualmente si ya no trabaja contigo

## ¿Por qué tener este sistema?

| Sin sistema                                         | Con este MVC                               |
|-----------------------------------------------------|---------------------------------------------|
| Dependes de alguien para cada cambio en la carta    | Tú mismo actualizas productos y precios     |
| Tus clientes no encuentran tu menú en internet      | Tu carta está siempre disponible y actualizada |
| Pierdes el rastro de quién accede al panel          | Control total de usuarios y roles           |
| Sin acceso remoto al negocio                        | Administras desde cualquier lugar           |
| Datos sueltos en varios lados                       | Todo centralizado en una base de datos      |
